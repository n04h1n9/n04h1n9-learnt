# 基础知识和基本数据类型  

## 基础知识  

- python语言特点：简单、强大、运行效率低  

- 安装python环境：linux自己给我装有，ide用的是neovim+lazyvim  

- 打开python的方式(linux)：终端输入python,或用创建一个.py后缀的文件夹，用ide写  

- 要在终端打印内容，输入`print('内容')`,如果括号内什么都没有（包括引号），则换行

- 单行注释一般用#，多行注释一般用三个引号（一般是双引号）将整个代码包起来。在成熟的ide中，一般都有选定一段代码块注释一整块的功能，在vim中是可视模式选定后,依次摁下gcc，快速注释单行代码可以依次摁下gcc  

- 变量是编程语言中数据的载体,变量的值可以读取或修改  

- 计算机能处理很多类型的数据，python语言中内置了很多种数据类型，我们也可以自己定义数据类型，最常用的数据类型有以下几种：  

  - 整型(int)可以处理任意大小的整数,并且支持二进制、四进制、八进制、十进制和十六进制的**整数**  

  - 浮点型(float)可以处理小数，为什么叫浮点数，是因为如果用科学记数法表示浮点数时，有效数字（前面的小数）会浮动，这么说说不清楚，以下是~~实机演示~~:  
  11.4514 = 1.14514 x 10 = 114.514e-1

  - 字符串型(str)是用单引号或双引号包裹起来的任意文本,上面的`print('内容')`括号里面的就是字符串。关于字符串的高级内容，在容器型数据类型会讲

  - 布尔型(bool)只有`True`和`False`两种值，叫做布尔值，当一个变量只能取到两种值时，可以用布尔型

- 变量命名规则：可以用数字，unicode字符（万国码，包含了世界上绝大多数字符，但建议还是用英文）或下划线，数字不能开头,大小写敏感，且不能与python中的保留字（就函数、关键字这些）重名。惯例上变量用英文，多个单词用下划线连接，关于惯例，这里有个[电子榨菜](https://www.bilibili.com/video/BV1FXBCBnELM/ "变量的名义")

- 变量的运算符参见[python字典](/python_note/python%E5%AD%97%E5%85%B8/1.%E5%8F%98%E9%87%8F%E8%BF%90%E7%AE%97%E7%AC%A6.md),以下是常见运算：  

```
a = 1 #赋值运算
a + 1 #算术运算
a = a + 1 #赋值运算，当a为1时，输出2
a += 1 #同上
b = 2
a == b #比较运算，输出布尔值
c = True
d = False
c and d #逻辑运算,输出布尔值
```

- 变量类型可以用type(变量名)来检查

- 有关变量类型的转换及其演示，参见[python字典](/python_note/python%E5%AD%97%E5%85%B8/2.%E5%8F%98%E9%87%8F%E7%B1%BB%E5%9E%8B%E8%BD%AC%E6%8D%A2.md)
