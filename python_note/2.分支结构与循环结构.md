# 分支结构与循环结构

## 分支结构  

### if-else关键字结构  

演示如下：  

```python
if 条件1:
    要执行的代码或代码块
elif 条件2:
    要执行的代码或代码块
else:
    条件都不成立时执行的代码或代码块
```

> python里的缩进一般为四格，用tab键可以快速缩进四格，缩进用于明确那段代码是那个关键字管的，python对缩进的代码很严格

### match-case关键字结构  

演示如下：

```python
match 一个东西:
    case 结果1: 运行这个代码块
    case 结果2: 运行这个代码块
```

> 分支多的时候用match-case比较好

## 循环结构  

### for-in循环  

演示如下：

```python
for i in range(10): #当这个i没有参与代码块时，可以用下划线_代替
    代码块
```

> range函数用于构造一个序列，以下是用法实例：  

1. `range(101)`取到从0到100共101个数字,取不到101  

2. `range(1, 114)`取到1到113,左开右闭  

3. `range(1, 114, 2)`取到1,3,5......113,左开右闭，跨度为2

4. `range(114, 0, -2)`取到114,112,110......2,左开右闭，跨度为-2

### while循环  

演示如下：  

```python
while 条件: #条件输出布尔值，为真则循环，为假则跳出循环
    代码块
```

> 确定循环次数用for,不确定用while

## 嵌套结构和break,continue关键字  

分支结构和循环结构可以嵌套，即某结构里再套一个结构  以下是实例：

```python
i = 1
while True:
    i += 1 
    if i % 2 != 0:
        continue
    if i == 50:
        break
```

其中的if-continue表示达成某条件后,跳过这次循环，执行下一循环;if-break表示达成某条件后，跳出循环
