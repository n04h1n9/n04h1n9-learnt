由于变量的方法种类繁多难记，所以单独写出来，变量的方法用于增删改查，这里会一一列出来，方法、函数后面要加括号

## 列表

由于列表有序可变，所以增删改查都可以

- 增

`items.append(x)`在列表末尾追加元素

`items.insert(1,x)`在索引值为1的地方插入元素x

- 删

`items.remove(x)`指定删除x元素

`items.pop(1)`删除索引位置为1的元素，参数为空则删除最后一个元素

`items.clear()`清空列表元素

- 改

`items.sort()`列表排序

`items.reverse()`元素反转

`items.clear()`清空列表元素

- 查

`items.index(x)`查找x元素的位置

`items.count(x)`查找x元素的频次

## 元组

由于元组除不可变外跟列表没有区别，所以元组只能用列表方法中的查,这里不再赘述

## 集合

由于集合无序，所以不能改查，又由于集合无序，所以增删方法跟前两者不完全一致

- 增

`set.add(x)`添加元素x

- 删

`set.discard(x)`删除元素x

`set.remove(x)`删除元素x

`set.pop()`删除随机元素，会返回被删元素

不可变集合连删改都不行

## 字典

- 增

`x.update(y)`用y字典覆盖x字典，也可用`x | y`运算符实现

- 删

`dict.pop(key)`删除键值对，返回键

`dict.popitem(key)`删除键值对，返回键值二元组

`dict.clear()`清空字典

- 改

`dict.items()`将键值组成二元组

- 查

`dict.get(key)`通过键获取对应的值

`dict.key()`获取所有键

`dict.values()`获取所有值

## 字符串

容器方法难度的很大一部分来源于字符串，虽然字符串不能增删，但改查也够受了

- 改（输出值，不改变原字符串）

  - 大小写相关

  `s.capitalize()`首字母大写

  `s.title()`每个单词首字母大写

  `s.upper()`每个字母都大写

  `s.lower()`每个字母都小写

  - 格式化

  `s.center(20, '*')`左右各10个*，使字符串居中，如果无第二个参数，则左右两边为棍母

  `s.rjust(20)`使字符串靠右，参数规则同上

  `s.ljust(20, ~)`右边20个~，使字符串靠左，参数规则同上

  `'-33'.zfill(5)`在字符串左侧补0，补够5位，负号单独放出来,输出-00033

  ```python
  '{0} * {1} = {2}'.format(a, b, a * b)
  #或
  f'{a} * {b} = {a * b}
  #输出a * b = a * b
  ```
  
  - 修建操作

  `s.strip('x')`剪掉字符串两端多余的x(如果括号为空则修建棍母)，此外还有`lstrip`和`rstrip`方法

  - 拆分与合并

  `s.split('#', 2)`将一字符串拆分进一个列表，默认按空格进行拆分，可制定次数

  `s.join([a])`以s为纽带，将列表内的字符串合并

  - 编解码

  ```python
  a = 'hyw'
  b = a.encode(utf-8) #编码，将字符串变为字节串
  d = c.decode(gbk) #解码，与编码相反
  ```

- 查

  - 查找

  `s.find('hyw', 1)`第二个参数没有就是查找索引位置；有的话就是判断位置，有错返回-1

  `s.index('hyw', 1)`同上，有错直接报错

  - 性质判断（输出布尔值）

  `s.startwith('x')`是否以某字符开头

  `s.endwith('x')`是否以某字符串结尾

  - is开头方法

    `s.isdigit()`判断字符串是否完全由数字组成

    `s.isalpha()`判断字符串是否完全由unicode字符（不含emoji）组成

    `s.alnum()`判断字符串是否由字母和数字组成
